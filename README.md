# Load-habitat-satellite-images
Python tool for processing satellite images for wildlife habitat analysis

## Project Description
Python-based tool for processing forest and satellite imagery to support wildlife corridor identification and habitat connectivity analysis.

## Features
- Automated image loading and processing
- Image enhancement for better habitat analysis
- Brightness and contrast optimization
- Professional image output generation
